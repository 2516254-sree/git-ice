FRONT END TOOLS

    In web development, tools are categorised into the front end (what users see and interact with) and the back end (the server-side logic and data management). Some tools, like Visual Studio Code and JavaScript, are versatile enough to be used across both layers.

    Front-End Tools (The "Client-Side") Front-end development focuses on the user interface (UI) and user experience (UX).

    Core Languages:

    HTML:

    Defines the structure and skeleton of a webpage.

    CSS:

    Controls the visual style, including layout, colours, and fonts.

    JavaScript:

    Adds interactivity and dynamic features like animations or form validation.

    Frameworks & Libraries:

    React:

    A popular library by Meta for building interactive UI components.

    Angular:

    A robust framework by Google for large-scale applications.

    Vue.js:

    A versatile, simple-to-learn framework for high-performance apps.

    Tailwind CSS:

    A utility-first CSS framework for rapid styling.

    Design & Debugging:

    Figma:

    A collaborative tool for UI/UX design and prototyping.

    Chrome DevTools:

    Built-in browser tools for real-time testing and debugging.
